Introduction to HTML
1. What is HTML?

HTML stands for HyperText Markup Language.

It is the standard language used to create and structure web pages. Every website you see on the internet is built using HTML.

Note: HTML is not a programming language. It is a markup language that tells the browser how to display content.
2. How Does the Web Work?

When you open a website, this happens behind the scenes:

1 You type a URL like google.com in your browser.
2 Browser sends a request to the web server where the website is stored.
3 Server sends back HTML files along with CSS and JavaScript.
4 Browser reads the HTML and displays the web page on your screen.
3. Role of HTML in Web Development

HTML is the skeleton of every website. It defines the structure and content.

What HTML Does:

✅ Structure: Creates headings, paragraphs, lists, tables

✅ Content: Adds text, images, videos, links

✅ Forms: Creates input fields, buttons, dropdowns

✅ Semantics: Tags like <header> <footer> <article> give meaning

Important: HTML handles structure only. For styling we use CSS, and for functionality we use JavaScript.
4. HTML, CSS & JavaScript - The Web Trio
TechnologyRoleExample
HTMLStructureSkeleton of page
CSSStylingColors, fonts, layout
JavaScriptFunctionalityClick events, animations

Think of it like a human body: HTML is the skeleton, CSS is the skin/clothes, JavaScript is the brain.

5. Basic HTML Example

Here is a simple HTML page structure:

Example: Basic HTML
<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first website.</p>
</body>
</html>

Output: Browser will display "Hello World" as a heading and "This is my first website" as a paragraph.

6. Why Learn HTML?

✅ Foundation: You cannot build websites without HTML

✅ Easy to Learn: Simple tags, beginner friendly

✅ Career: First step to become a web developer

✅ Control: Build your own blog, portfolio, or business site

Fun Fact: The first website ever created in 1991 is still online at info.cern.ch. It was built using only HTML!
