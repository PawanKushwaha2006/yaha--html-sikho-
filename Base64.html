Use of Base64 - For BCA

1. Web Tech Practical: Display image in HTML without file → <img src="data:image/png;base64,...">

2. API/JSON: To send file to server → Convert to Base64 string and put in JSON

3. JWT Token: Token received after login → It is Base64 encoded

4. Email: Attachment is converted to Base64 and sent

Important: Base64 ≠ Encryption. It's only encoding. Don't use it for passwords.
