HTML Skeleton Structure
1. Basic Skeleton Structure

Every HTML page has a basic skeleton. These 4 tags are mandatory:

Complete Skeleton:
<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>
Note: This structure is like a human body. Without skeleton, body cannot stand. Same way, without these tags, HTML page will not work properly.
2. <!DOCTYPE html> - Document Type Declaration
<!DOCTYPE html>

Purpose: Tells the browser that this is an HTML5 document.

Position: Must be the very first line of your HTML file.

Why Important: Without this, browsers go into "Quirks Mode" and your page may look broken.

Common Mistake: Writing <DOCTYPE> or forgetting !. It must be exactly <!DOCTYPE html>
3. <html> - The Root Element
<html> </html>

Purpose: This is the root/container of the entire HTML document. All other tags go inside this.

Attribute: lang="en" tells browsers and screen readers the language of the page.

Why Important: Browser knows where your HTML code starts and ends.

Example with lang attribute:

<html lang="en"> for English
<html lang="hi"> for Hindi
4. <head> - The Brain of the Page
<head> </head>

Purpose: Contains meta-information about the page. This content is NOT visible on the web page.

Contains: Title, CSS links, JavaScript links, meta tags, favicon, fonts

Why Important: SEO, page title in browser tab, mobile responsive settings all go here.

Common Tags Inside <head>:

TagPurpose
<title>Page title shown in browser tab
<meta charset="UTF-8">Character encoding for Hindi, emoji support
<meta name="viewport">Makes site mobile responsive
<link>Connect CSS files, favicon
<style>Internal CSS code
<script>JavaScript code
5. <body> - The Visible Content
<body> </body>

Purpose: Contains all the content that users see on the web page.

Contains: Headings, paragraphs, images, videos, buttons, forms - everything visible

Why Important: Without body tag, browser won't display anything to users.

Example:

Content Inside Body:
<body>
  <h1>Welcome</h1>
  <p>This is visible to users</p>
  <img src="photo.jpg">
</body>
6. Summary - Skeleton Comparison
TagVisible?RoleAnalogy
<!DOCTYPE>NoDefines HTML versionBirth Certificate
<html>NoRoot containerFull Body
<head>NoMeta info, settingsBrain
<body>YesVisible contentFace + Body
Remember: Head = Invisible information for browser. Body = Visible content for users.
