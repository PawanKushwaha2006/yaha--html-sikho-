Chapter 13: Advanced Tables

1. Table Caption - <caption>

The <caption> tag defines a title for the table. It must be inserted immediately after the <table> tag.

<table>
  <caption>Monthly Sales Report 2026</caption>
  <tr>
    <th>Month</th>
    <th>Sales</th>
  </tr>
  <tr>
    <td>January</td>
    <td>\$5000</td>
  </tr>
</table>

Output:

Monthly Sales Report 2026
Month Sales
January $5000
February $6500

2. Colspan - Merge Columns

colspan attribute merges multiple columns into one cell. It spans horizontally.

<table>
  <tr>
    <th colspan="3">Student Information</th>
  </tr>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Grade</th>
  </tr>
  <tr>
    <td>Alex</td>
    <td>16</td>
    <td>A</td>
  </tr>
</table>

Output:

Student Information
Name Age Grade
Ragni 19 A
Rohit 20 B+

3. Rowspan - Merge Rows

rowspan attribute merges multiple rows into one cell. It spans vertically.

<table>
  <tr>
    <th>Department</th>
    <th>Employee</th>
    <th>Salary</th>
  </tr>
  <tr>
    <td rowspan="2">Sales</td>
    <td>John</td>
    <td>\$5000</td>
  </tr>
  <tr>
    <td>Sarah</td>
    <td>\$5500</td>
  </tr>
</table>

Output:

Department Employee Salary
Mechanical Sonu $5000
Pawan $5500
IT Amit $7000
Rahul $7500

4. Combining Colspan & Rowspan

<table>
  <caption>Exam Schedule</caption>
  <tr>
    <th rowspan="2">Day</th>
    <th colspan="2">Time</th>
  </tr>
  <tr>
    <th>Morning</th>
    <th>Evening</th>
  </tr>
  <tr>
    <td>Monday</td>
    <td>Math</td>
    <td>Physics</td>
  </tr>
</table>

Output:

Exam Schedule
Day Time
Morning Evening
Monday Math Physics
Tuesday Chemistry Biology

5. Advanced Example: Complex Table

<table>
  <caption>Company Quarterly Report</caption>
  <tr>
    <th rowspan="2">Region</th>
    <th colspan="4">Sales (in \$)</th>
  </tr>
  <tr>
    <th>Q1</th>
    <th>Q2</th>
    <th>Q3</th>
    <th>Q4</th>
  </tr>
  <tr>
    <td>North</td>
    <td>10K</td>
    <td>12K</td>
    <td>15K</td>
    <td>18K</td>
  </tr>
  <tr>
    <td colspan="5">Total: 55K</td>
  </tr>
</table>
✅ Important Notes:
1. colspan="n" - Cell spans 'n' columns horizontally
2. rowspan="n" - Cell spans 'n' rows vertically
3. <caption> always comes right after <table> tag
4. Count carefully - total cells in each row must match after spanning
5. Use CSS to style captions: caption-side: top or bottom

6. Common Mistakes ❌

<!-- Wrong: Caption after rows -->
<table>
  <tr><td>Data</td></tr>
  <caption>Title</caption> <!-- Wrong position -->
</table>

<!-- Wrong: Colspan exceeds column count -->
<tr>
  <td colspan="5">Text</td> <!-- But table only has 3 columns -->
</tr>
