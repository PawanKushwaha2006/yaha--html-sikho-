Chapter 8: Working with Images

HTML Images <img> Tag

The <img> tag is used to add images to a web page. It is a self-closing tag - it does not have a closing tag.

Basic Syntax:

<img src="image.jpg" alt="Description of image">

1. src Attribute - Most Important

src = source. It specifies the path or URL of the image.

<!-- Image in the same folder -->
<img src="photo.jpg" alt="My Photo">

<!-- Image in a subfolder -->
<img src="images/logo.png" alt="Company Logo">

<!-- Online image URL -->
<img src="https://example.com/banner.jpg" alt="Banner">

2. alt Attribute - SEO + Accessibility

alt = alternative text. If the image fails to load, this text is displayed. It is also important for screen readers.

<img src="cat.jpg" alt="A cute white cat sleeping on sofa">
💡 Pro Tip: Always write the alt attribute. It helps with Google SEO and is essential for visually impaired users.

3. Resizing Images

Method 1: Width & Height Attributes - Directly in HTML

<!-- Fixed size in pixels -->
<img src="photo.jpg" alt="Photo" width="300" height="200">

<!-- Only specify width, height adjusts automatically -->
<img src="photo.jpg" alt="Photo" width="400">

Method 2: CSS Style - Best Way

<img src="photo.jpg" alt="Photo" style="width:100%; max-width:500px; height:auto;">

<!-- Responsive image -->
<img src="photo.jpg" alt="Photo" style="width:100%; height:auto;">

4. Important Image Attributes

<img
  src="banner.jpg"
  alt="Website Banner"
  width="800"
  height="400"
  title="Click to view full size"
  loading="lazy"
>
  • title - Shows a tooltip on mouse hover
  • loading="lazy" - Page loads faster, image loads when scrolled into view
  • width & height - Sets size, prevents layout shift

5. Making an Image a Link

<a href="https://google.com" target="_blank">
  <img src="google-logo.png" alt="Google" width="100">
</a>

6. Common Mistakes ❌

<!-- Wrong: alt missing -->
<img src="photo.jpg">

<!-- Wrong: Adding a closing tag -->
<img src="photo.jpg"></img>

<!-- Wrong: Both height and width fixed, image will stretch -->
<img src="photo.jpg" width="100" height="500">

7. Best Practices ✅

✅ Always Do:
1. Always write the alt attribute
2. Compress images - Use TinyPNG
3. Add loading="lazy" for large images
4. Define width and height - prevents layout shift
5. Use style="width:100%;height:auto;" for responsiveness

8. Image Formats

  • .jpg /.jpeg - Best for photos, small size
  • .png - For transparent backgrounds
  • .svg - For logos and icons, stays sharp on zoom
  • .webp - Modern format, smallest size + best quality
