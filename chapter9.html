Chapter 9: Unordered Lists - Bullet Points

The <ul> tag creates an unordered list. Each item uses <li> tag. Browser shows bullets by default.

1. Basic Syntax

<ul>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ul>
Output:
  • First Item
  • Second Item
  • Third Item
Key Rules:
1. <ul> = Unordered List - Container for all items
2. <li> = List Item - Each bullet point
3. <li> must always be inside <ul> or <ol>
4. Order doesn't matter in <ul> - that's why "unordered"

2. Real World Examples

Example 1: Navigation Menu

<ul>
  <li><a href="#home">Home</a></li>
  <li><a href="#about">About</a></li>
  <li><a href="#contact">Contact</a></li>
</ul>
Output:
  • Home
  • About
  • Contact

Example 2: Features List

<h3>Course Features:</h3>
<ul>
  <li>30 Detailed Chapters</li>
  <li>Live Code Examples</li>
  <li>Mobile Friendly</li>
  <li>Certificate Included</li>
</ul>
Output:

Course Features:

  • 30 Detailed Chapters
  • Live Code Examples
  • Mobile Friendly
  • Certificate Included

3. Nested Lists - List Inside List

You can put a <ul> inside an <li> to create sub-items.

<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>Backend
    <ul>
      <li>Node.js</li>
      <li>Python</li>
    </ul>
  </li>
</ul>
Output:
  • Frontend
    • HTML
    • CSS
    • JavaScript
  • Backend
    • Node.js
    • Python
Important: Nested <ul> must be inside an <li>. Never put <ul> directly inside another <ul>.

4. Changing Bullet Style with CSS

Use list-style-type property to change bullets.

CSS ValueOutputCode
disc● Default filled circleul { list-style-type: disc; }
circle○ Hollow circleul { list-style-type: circle; }
square■ Filled squareul { list-style-type: square; }
noneNo bulletul { list-style-type: none; }
Different Bullet Styles:

disc - Default:

  • HTML
  • CSS

circle:

  • HTML
  • CSS

square:

  • HTML
  • CSS

none - No bullets:

  • HTML
  • CSS

5. Custom Bullets with Emojis/Icons

Remove default bullets and add your own using ::before

ul {
  list-style-type: none;
  padding-left: 0;
}
ul li::before {
  content: "✅ ";
  margin-right: 8px;
}
Custom Emoji Bullets:
  • Task Completed
  • Project Submitted
  • Certificate Earned
Custom Arrow Bullets:
  • Step 1: Install VS Code
  • Step 2: Create index.html
  • Step 3: Write your code

Wrong nesting<ul><ul><li>Item</li></ul></ul><ul><li><ul><li>Item</li></ul></li></ul>

7. ul vs ol - When to Use What?

TagFull NameWhen to UseExample
<ul>Unordered ListOrder doesn't matterFeatures, Menu items, Ingredients
<ol>Ordered ListOrder matters, stepsRecipe steps, Rankings, Instructions
Pro Tip: Use <ul> for navigation menus, feature lists, tags. Use <ol> for tutorials, rankings, step-by-step guides.

9. Summary - Golden Rules

  1. Structure: <ul> is container, <li> is each item
  2. Order doesn't matter: That's why called "unordered"
  3. Nesting: Put <ul> inside <li> for sub-lists
  4. Styling: Use CSS list-style-type to change bullets
  5. Semantics: Use for actual lists, not for layout. Use <p> for paragraphs
  6. Navigation: Most website menus are built with <ul> + <li> + <a>
