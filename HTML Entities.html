©️ Chapter: HTML Entities

HTML Entities are special codes used to display reserved characters and symbols that either have special meaning in HTML or aren't on your keyboard. They always start with & and end with ;

1. Why Do We Need HTML Entities?

Some characters are reserved in HTML. If you type them directly, the browser gets confused.

Problem Example:

<p>To create a tag use <div> </p>

Browser Output: To create a tag use (The <div> disappears because browser thinks it's a real tag)

Correct Solution:

<p>To create a tag use &lt;div&gt; </p>

Browser Output: To create a tag use <div>

2. Most Common HTML Entities

Symbol Description Entity Name Entity Number Use Case
< Less than &lt; &#60; Display < in text
> Greater than &gt; &#62; Display > in text
& Ampersand &amp; &#38; Display & in text
" Double quote &quot; &#34; Inside attribute values
' Single quote &apos; &#39; Inside attribute values
  Non-breaking space &nbsp; &#160; Prevent line breaks
© Copyright &copy; &#169; Footer copyright
® Registered trademark &reg; &#174; Brand names
™ Trademark &trade; &#8482; Product names

7. Non-Breaking Space &nbsp;

Regular spaces collapse in HTML. Multiple spaces become one. &nbsp; forces spaces to display and prevents line breaks.

Example:

<p>Hello     World</p>
<!-- Output: Hello World (extra spaces removed) -->

<p>Hello&nbsp;&nbsp;World</p>
<!-- Output: Hello     World (spaces preserved) -->

Live Demo:

Without nbsp: 10km

With nbsp: 10 km (km won't wrap to next line alone)

Use Case: Keep "Mr. Smith" together. Keep "100 km" together. Prevent awkward line breaks.

8. Entity Name vs Entity Number

Every symbol has two formats:

  • Entity Name: &copy; - Easy to remember
  • Entity Number: &#169; - Works in all browsers, even old ones
<p>Copyright &copy; 2026</p>
<p>Copyright &#169; 2026</p>
<!-- Both output: Copyright © 2026 -->
Best Practice: Use entity names for common symbols. Use numbers for rare symbols that don't have names.

9. Complete Practical Example

<footer>
    <p>&copy; 2026 MyCompany&reg; All rights reserved.</p>
    <p>Price: &#8377;999 &plusmn; &#8377;50</p>
    <p>Use &lt;div&gt; tag for layout</p>
    <p>Temperature: 25&deg;C &rarr; 30&deg;C</p>
    <p>Rating: &star;</p>
</footer>

Browser Output:

© 2026 MyCompany® All rights reserved.

Price: ₹999 ± ₹50

Use <div> tag for layout

Temperature: 25°C → 30°C

Rating: ★★★★★

⚠️ Common Mistakes

Mistake #1: Forgetting Semicolon

<!-- WRONG: Missing ; -->
&copy

<!-- CORRECT -->
&copy;

Mistake #2: Using & Directly

<!-- WRONG: & alone is invalid -->
<p>Tom & Jerry</p>

<!-- CORRECT -->
<p>Tom &amp; Jerry</p>

Mistake #3: Not Escaping < and > in Code

<!-- WRONG: Browser thinks it's a tag -->
<code><div></code>

<!-- CORRECT -->
<code>&lt;div&gt;</code>

🛠️ Practice Task

Create a footer with: 1) Copyright symbol with current year 2) Registered trademark symbol 3) Price in Indian Rupees: ₹1,499 4) Temperature range: 20°C → 35°C 5) Text: "Use <section> tag"

Example Solution:

<footer>
    <p>&copy; 2026 WebDev&reg;</p>
    <p>Price: &#8377;1,499</p>
    <p>Temperature: 20&deg;C &rarr; 35&deg;C</p>
    <p>Use &lt;section&gt; tag for content</p>
</footer>
