Paragraphs & Text Formatting
1. <p> - Paragraph Tag
<p> </p>

Purpose: Creates a paragraph. Browsers automatically add space before and after each paragraph.

Block-level: It always starts on a new line and takes full width.

Example:

<p>This is first paragraph.</p>
<p>This is second paragraph.</p>
OUTPUT:

This is first paragraph.

This is second paragraph.

Note: Multiple spaces or line breaks in HTML code are ignored. Browser shows only 1 space. Use <br> for line break inside paragraph.
2. <b> vs <strong> - Bold Text
<b> </b>

Purpose: Makes text bold visually. No extra importance for screen readers or SEO.

Use when: You just want bold styling without meaning.

<strong> </strong>

Purpose: Makes text bold + tells browser this text is important.

Use when: Text has semantic importance. Better for SEO and accessibility.

Example:

<p>This is <b>bold</b> text.</p>
<p>This is <strong>important</strong> text.</p>
OUTPUT:

This is bold text.

This is important text.

✓ SEO Tip: Use <strong> for keywords and important points. Google gives more weight to it than <b>.
3. <i> vs <em> - Italic Text
<i> </i>

Purpose: Makes text italic visually. Used for technical terms, foreign words, thoughts.

Use when: Just styling, no emphasis needed.

<em> </em>

Purpose: Makes text italic + adds emphasis. Screen readers change voice tone.

Use when: You want to stress or emphasize a word.

Example:

<p>The word <i>HTML</i> means HyperText Markup Language.</p>
<p>You <em>must</em> complete this task.</p>
OUTPUT:

The word HTML means HyperText Markup Language.

You must complete this task.

4. <u> - Underline Text
<u> </u>

Purpose: Adds underline to text. Used for misspelled words, proper names in Chinese, etc.

Example:

<p>This is <u>underlined</u> text.</p>
OUTPUT:

This is underlined text.

✗ Warning: Avoid using <u> for normal text. Users think underlined text is a link. Use CSS text-decoration: underline; if you need it for design.
5. <mark> - Highlight Text
<mark> </mark>

Purpose: Highlights text with yellow background like a marker. Shows relevance in search results.

Use when: You want to highlight search terms or important parts.

Example:

<p>HTML is the <mark>foundation</mark> of web development.</p>
OUTPUT:

HTML is the foundation of web development.

Note: Default color is yellow. You can change it with CSS: mark { background: lightblue; }

7. Combined Example

Using All Tags Together:

<p>
  <strong>HTML</strong> is a <em>markup language</em>.
  It uses <b>tags</b> to define <i>elements</i>.
  The most <mark>important</mark> tag is <u><html></u>.
</p>
OUTPUT:

HTML is a markup language. It uses tags to define elements. The most important tag is <html>.
